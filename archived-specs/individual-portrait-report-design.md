# 1. Report Philosophy

## Design Principles

### 1.1 Narrative Over Data

The Portrait is not a test result. It's a story about who you are in relationships.

```
NOT THIS:
"Your ECR-R Anxiety subscale score is 5.2 (high)."

THIS:
"You carry a deep capacity for connection and a keen attunement 
to relationship dynamics. Your heart is oriented toward closeness."
```

### 1.2 Strength-Based Framing

Every pattern is presented as an adaptive strategy that made sense, even when it now causes problems.

```
NOT THIS:
"You have an anxious attachment style, which causes 
clingy behavior and relationship problems."

THIS:
"When you sense distance, you move toward—seeking connection,
wanting to resolve. This strategy developed because connection 
matters deeply to you. The challenge is that it can sometimes 
feel like pressure to your partner."
```

### 1.3 Actionable Insight

Every insight connects to what the user can do differently.

```
Pattern → Cost → Invitation → Practice → Anchor
```

### 1.4 Paced Revelation

Deep content is progressive. Surface first, depth on demand.

```
LAYER 1: Summary card (always visible)
LAYER 2: Expanded explanation (tap to reveal)
LAYER 3: Deepening questions (optional exploration)
```

# 4. Complete Portrait Flow

## 4.1 Emotional Pacing

```
SECTION              EMOTIONAL TONE           PURPOSE
─────────────────────────────────────────────────────────
Cover                Warm, personal           Ground in identity
Introduction         Orienting, calm          Set expectations
Snapshot             Curious, overview        Big picture
Attachment           Tender, validating       Core understanding
Parts                Curious, normalizing     Complexity acknowledged
Regulation           Practical, grounding     Body awareness
Values               Aspirational, connected  Heart orientation
Negative Cycle       Honest, compassionate    Pattern awareness
Growth Edges         Hopeful, challenging     Forward motion
Anchors              Supportive, practical    Tools for hard moments
Partner Guide        Vulnerable, brave        Relational bridge
Deepening            Reflective, spacious     Optional depth
Closing              Encouraging, warm        Next steps

```
SECTION              LENGTH      EXPANDABLE?
───────────────────────────────────────────
